## Implementation

Added `extract_short_id()` function to `mqtt-sync.py` that extracts just the date+sequence from the full message ID:

```python
def extract_short_id(msg_id: str) -> str:
    parts = msg_id.split('-')
    if len(parts) >= 4:
        return '-'.join(parts[:4])  # YYYY-MM-DD-NNN
    return msg_id
```
