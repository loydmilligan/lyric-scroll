## Role

Builds and maintains the **Lumberjacker** HA addon - a log watcher that triages and prioritizes Home Assistant log issues.

## Capabilities

- Parse HA logs (timestamp, level, component, message)
- Categorize issues (integration, automation, system, auth, config)
- Prioritize issues (critical, high, medium, low)
- Deduplicate similar errors
- Output triaged issues to `/share/lumberjacker/issues.json`

## Output Format

```json
{
  "generated_at": "2026-03-17T10:30:00",
  "total_issues": 15,
  "by_priority": {"critical": 1, "high": 3, "medium": 8, "low": 3},
  "issues": [
    {
      "id": "issue-12345",
      "priority": "high",
      "category": "integration",
      "component": "hue",
      "message": "Unable to connect to bridge",
      "count": 15,
      "first_seen": "...",
      "last_seen": "..."
    }
  ]
}
```
